# BazaarDB.gg Guide

## What It Offers
- Complete item/skill database
- Advanced search syntax
- Enchantment breakdowns
- Merchant pools & trainer offerings
- Tier lists
- Patch notes tracking

## Search Syntax
Use filters to find specific items:
- **Hero**: `hero:vanessa`, `hero:dooley`
- **Size**: `size:small`, `size:medium`, `size:large`
- **Tier**: `tier:bronze`, `tier:silver`, `tier:gold`, `tier:diamond`, `tier:legendary`
- **Tags**: `tag:weapon`, `tag:tech`, `tag:tool`, `tag:dragon`

1. Check tier lists after each patch
2. Look up items you encounter in runs
3. Study merchant pools to know what's available when
4. Use search to find synergies (e.g., all items with "haste")
