# PWA Icons

Create app icons at these sizes:

- 72x72
- 96x96
- 128x128
- 144x144
- 152x152
- 192x192
- 384x384
- 512x512

## Quick Generation

Use a service like:
- https://www.pwabuilder.com/imageGenerator
- https://realfavicongenerator.net/

Upload a 512x512 PNG logo and it will generate all sizes.

## Design Guidelines

- Use the Bazaar Companion logo or "BC" monogram
- Background: #0f0f23 (dark theme color)
- Foreground: #16C79A (accent color)
- Include some padding for maskable icons
